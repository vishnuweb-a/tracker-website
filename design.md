# 🎨 StudySprint Website Design System

> Minimal • Modern • Product Showcase • White + Yellow Theme

---

# Design Philosophy

StudySprint is built for students.

The website should feel:

- Clean
- Motivating
- Bright
- Fast
- Premium
- Focused
- Productivity Driven

Avoid gaming aesthetics.

Avoid overly colorful gradients.

The UI should communicate:

> "Study smarter. Stay consistent."
